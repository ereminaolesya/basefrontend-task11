## Описание

Реализация HTML+CSS+JavaScript проект интерактивного счетчика с множественными способами изменения значения.
Задача взята из учебного практического курса.

## Требования к задаче
Основная функциональность:
Система изменения значений
Кнопки быстрого изменения: -3, -2, -1, +1, +2, +3 для пошагового управления
Произвольное значение: input поле с валидацией для установки любого числа
Сброс счетчика: кнопка Reset возвращает значение к 0
Ограничения диапазона: значения ограничены пределами ±1e9 (±1,000,000,000)

Требования к реализации текстового поля:
Текстовое поле ввода произвольного значения должно быть оформлено в виде HTML-формы
Отправка формы должна работать на клавишу Enter и наэкранную кнопку Apply
Форма должна поддерживать следующую валидацию
Можно вводить только числа
Запрещен ввод дробных чисел
Допустимый диапазон: [-10.000.000; 10.000.000]
Запрещен ввод пустого значения (поле должно быть обязательно для заполнения)
После отправки формы значение внутри текстового поля должно сброситься

Требования к отображению:
Числовое значение должно менять цвет в зависимости от знака числа
Не нужно реализовать адаптивность приложения под мобильные разрешения
